## A jornada de modernizacao do HCL Domino

Depois de mais de duas decadas construindo solucoes empresariais sobre a plataforma Lotus Notes/Domino, chegou o momento de dar o proximo passo: modernizar sem abandonar o que funciona.

Este blog documenta a nossa jornada real de construcao de uma aplicacao moderna utilizando **Spring Boot** e **Vaadin**, integrada ao **HCL Domino** por meio da **Domino REST API (DRAPI)**. Nao se trata de uma reescrita do zero, mas de uma evolucao arquitetural que respeita o investimento existente em dados, regras de negocio e, principalmente, no modelo de seguranca robusto que o Domino oferece.

### O que voce vai encontrar aqui

- **Autenticacao e autorizacao** baseadas na infraestrutura de seguranca do HCL Domino, com LDAP e tokens JWT gerados pelo DRAPI
- **Integracoes praticas** entre Spring Boot e Domino REST API, incluindo acesso a documentos, views e agentes
- **Desenvolvimento de interfaces** com Vaadin Flow, conectando componentes Java diretamente ao backend
- **Desafios reais** de migracao de um ERP corporativo com multiplos modulos integrados
- **Compliance tributario brasileiro** preservado durante a transicao (NFe, ICMS, PIS/COFINS e demais obrigacoes)

### Para quem e este blog

Se voce administra, desenvolve ou mantem aplicacoes HCL Domino e esta avaliando caminhos de modernizacao, este espaco e para voce. Compartilhamos cada descoberta, cada obstáculo e cada solucao que encontramos ao longo do caminho, com o objetivo de contribuir com a comunidade que enfrenta desafios semelhantes.

A ideia e simples: aprendemos algo novo, publicamos aqui. Assim, o conhecimento nao fica restrito a um projeto — ele se torna acessivel para todos que estao nesta mesma jornada.
